End-to-End MLOps Pipeline on AWS

Terraform EKS SageMaker CI/CD

Workshop Overview

Project Description

This workshop demonstrates a complete MLOps workflow on AWS for retail forecasting, featuring infrastructure as code, container orchestration, and automated ML pipelines.

Key Features:

  • Infrastructure as Code with Terraform
  • EKS with managed node groups for scalable inference API
  • SageMaker for ML training and model registry
  • ECR for Docker image management
  • CloudWatch for monitoring and alerting
  • Jenkins/Travis CI/CD pipelines

Architecture Components
