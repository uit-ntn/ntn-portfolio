Workshop Overview

Project Description

VinaShoes is a comprehensive AWS microservices e-commerce workshop featuring a modern shoe store built with cloud-native architecture and best practices.

Key Features:

  • Next.js frontend hosted on Amplify + CloudFront
  • NestJS microservices on ECS Fargate
  • DynamoDB for primary data storage
  • S3 + CloudFront for asset delivery
  • AWS WAF for security protection
  • Complete CI/CD with CodePipeline

Multi-Account Architecture

Production Account

  • • Next.js on Amplify
  • • API Gateway HTTP API
  • • ECS Fargate services
  • • DynamoDB + S3 storage
  • • VPC with endpoints

Security Account

  • • CloudTrail audit logging
  • • AWS Config compliance
  • • CloudWatch monitoring
  • • S3 Cross-Region backup

CI/CD Account

  • • CodeCommit repositories
  • • CodeBuild automation
  • • CodePipeline orchestration
  • • ECR container registry
